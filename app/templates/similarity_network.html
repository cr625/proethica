    How to Use This Network
    Interacting with the Network
    • Click a case (node) to see its details, outcome, and connections
    • Click a connection (edge) to see similarity breakdown and shared entities
    • Drag nodes to reposition; use zoom controls to navigate
    • Hover over a node to highlight its connections
    Layout Options
    • Force - Physics-based, similar cases cluster together
    • Circular - All cases arranged in a circle
    • Grid - Cases in a regular grid pattern
    • Radial - Grouped by outcome (ethical/unethical)
    Entity Overlap Filters

    The R, P, O, S, Rs, A, E, Ca, Cs buttons filter the network to show only cases sharing extracted entities of that D-tuple component type. Edge darkness indicates shared entity count.

    Shared Entities by Component

    Clicking any edge in the default similarity view loads a per-component breakdown of shared entities between the two cases. Each D-tuple component (Roles, Principles, Obligations, etc.) with matching entities is listed with its count and individual entity labels.
