ProEthica

An ethical decision-making simulator for professional domains with world-based scenarios and temporal reasoning.

ProEthica creates immersive ethical decision environments through customizable worlds and scenarios, evaluating decisions using a hybrid approach of rule-based validation, case-based reasoning, and guideline-based ethical assessment.

Explore Worlds Browse Scenarios
World Building

Create customizable worlds with domain-specific entities, guidelines, cases, and rulesets based on ontologies that define ethical frameworks.

Knowledge Integration

Integrates domain-specific knowledge and ethical guidelines using LangChain and Model Context Protocol (MCP) for interfacing with external models, data sources, and rules-based evaluation.

Ethical Decision Framework

Implements a hybrid ethical reasoning system combining rule-based validation, case-based reasoning, and principle-based assessment for comprehensive ethical evaluation.

Getting Started

To begin using ProEthica:

  1. Select an existing world or create a new one
  2. Explore existing scenarios or create a new one within your world
  3. Add characters, conditions, and resources to your scenario
  4. Create a timeline of events and actions
  5. Make decisions and see how they are evaluated

Professional Domains in Custom Worlds

ProEthica allows you to create custom worlds for any professional domain that has an associated code of ethics or standards of conduct or practice:

  • Use Existing Templates: Start with pre-configured worlds for various professional domains and ethical contexts
  • Create Custom Worlds for Domains: Define your own world which describes the professional domain with custom entities, guidelines, and rulesets
  • Import Ontologies: Use exisiting knowledge to populate the key features of the world. ontologies

Each world you create becomes a foundation for scenarios, with customizable entities, guidelines, cases, and rulesets tailored to the specific professional domain.
