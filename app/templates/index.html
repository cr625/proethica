ProEthica

An ethical decision-making simulator that evaluates professional decisions within customizable worlds using role-based guidelines and intelligent agents.

ProEthica creates immersive scenarios in ontology-defined worlds where specialized agents evaluate decisions using professional guidelines, contextual reasoning, and knowledge from integrated document sources.

World Building

Create customizable worlds with the system's help to define entities, roles, and relationships relevant to the target domain. Incorporate guidelines and cases that establish the ethical framework for evaluating decisions.

Knowledge Integration

Upload documents or provide URLs to guidelines and cases. The system processes these into a searchable knowledge base that agents can access during simulations to provide context-aware evaluations.

Ethical Decision Framework

Evaluates decisions against professional guidelines while recognizing justified exceptions. The system considers both the the guidelines and standards of a profession and relevant cases.

Getting Started

To begin using ProEthica:

  1. Select an existing world or create a new one
  2. Explore existing scenarios or create a new one within a world
  3. Add characters, conditions, and resources to a scenario. The system can assist by drafting descriptions and suggesting appropriate roles, conditions, resources, and actions from those defined in the world
  4. Create a timeline of events and actions that is considered in the ethical evaluation process.
  5. See recommendations for decisions and see how they were evaluated.

Professional Domains in Custom Worlds

ProEthica enables the creation of ontology-defined worlds for any professional domain, where decisions are evaluated based on the roles, guidelines, and entities that define the professional context:

  • Use Existing Templates: Start with pre-configured entity worlds for various professional domains.
  • Create Custom Worlds for Domains: Define your professional context through entities, their relationships, and the guidelines that govern decision-making
  • Import Documents and Knowledge: Upload professional guidelines, case studies, and other resources (PDF, Word, text, HTML) or provide URLs. The system will analyze these documents and apply the rules and cases to the decision evaluation.

Each ontology-defined world provides the context for scenarios where professional roles and their associated guidelines form the basis for ethical evaluation, supported by external tools and Zotero references through the MCP architecture.
