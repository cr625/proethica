About LangExtract Examples
What are LangExtract Examples?

LangExtract examples provide structured training data for few-shot learning in document analysis. Each example contains:

  • Input Text: Sample content to analyze
  • Expected Extractions: Structured output format with attributes
  • Domain Context: Professional domain specification
  • Section Type: Document section classification
Example Types
  • Factual: Key facts and entities extraction
  • Question: Ethical questions identification
  • Analysis: Ethical reasoning and principles
  • Conclusion: Decision outcomes and recommendations
  • Code Reference: Professional codes and standards
  • Dissenting: Alternative viewpoints and minority opinions
